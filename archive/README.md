# Archive

This directory contains retired code and test artifacts that are intentionally excluded from the active application and test runtime.

## Layout

- `legacy-src/`: backend modules preserved for historical reference only.
- `legacy-tests/`: retired backend test artifacts preserved for historical reference only.

The active backend auth runtime remains in `src/index.js`, `src/routes/auth.routes.js`, and `src/services/auth.service.js`.
